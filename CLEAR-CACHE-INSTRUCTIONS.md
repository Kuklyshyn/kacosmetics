# Інструкція по очищенню кешу

## Якщо ви не бачите зміни на сайті, виконайте наступні кроки:

### 1. Очистіть кеш браузера

#### Chrome / Edge

1. Натисніть `Ctrl + Shift + Delete` (Windows) або `Cmd + Shift + Delete` (Mac)
2. Виберіть "Зображення та файли в кеші"
3. Натисніть "Очистити дані"

**АБО**

Просто натисніть `Ctrl + Shift + R` (Windows) або `Cmd + Shift + R` (Mac) для жорсткого перезавантаження

#### Firefox

1. Натисніть `Ctrl + Shift + Delete`
2. Виберіть "Кеш"
3. Натисніть "Очистити зараз"

#### Safari

1. Натисніть `Cmd + Option + E`
2. Перезавантажте сторінку

### 2. Очистіть кеш WordPress

Якщо у вас встановлений плагін кешування (WP Super Cache, W3 Total Cache, WP Rocket тощо):

1. Перейдіть в адмін-панель WordPress
2. Знайдіть пункт меню вашого плагіну кешування
3. Натисніть "Clear Cache" або "Purge Cache"

### 3. Перевірте, чи активна тема

1. Перейдіть в `Appearance > Themes`
2. Переконайтеся, що активна тема `kacosmetics`

### 4. Перевірте файли

Переконайтеся, що всі файли на місці:

```
wp-content/themes/kacosmetics/
├── woocommerce/
│   ├── cart/
│   │   ├── cart.php
│   │   └── cart-totals.php
│   └── myaccount/
│       ├── my-account.php
│       ├── navigation.php
│       ├── dashboard.php
│       ├── form-edit-address.php
│       └── my-address.php
├── js/
│   ├── cart.js
│   └── my-account.js
└── style.css
```

### 5. Перевірте версію теми

Відкрийте файл `style.css` і перевірте версію:

```css
version: 1.3;
```

Якщо версія інша, файл не оновився. Завантажте його знову.

### 6. Режим інкогніто

Спробуйте відкрити сайт в режимі інкогніто:

- Chrome: `Ctrl + Shift + N`
- Firefox: `Ctrl + Shift + P`
- Safari: `Cmd + Shift + N`

Якщо в інкогніто все працює, проблема в кеші браузера.

### 7. Перевірте консоль браузера

1. Натисніть `F12` для відкриття Developer Tools
2. Перейдіть на вкладку "Console"
3. Перезавантажте сторінку
4. Перевірте, чи є помилки (червоні повідомлення)

### 8. Перевірте Network

1. Відкрийте Developer Tools (`F12`)
2. Перейдіть на вкладку "Network"
3. Перезавантажте сторінку
4. Знайдіть файли `style.css`, `cart.js`, `my-account.js`
5. Перевірте, чи вони завантажуються (статус 200)

### 9. Якщо нічого не допомагає

1. Деактивуйте всі плагіни
2. Активуйте стандартну тему WordPress
3. Активуйте знову тему `kacosmetics`
4. Активуйте плагіни по одному

### 10. Перевірка через FTP/SSH

Якщо у вас є доступ до сервера:

1. Підключіться через FTP або SSH
2. Перейдіть в `wp-content/themes/kacosmetics/`
3. Перевірте дату модифікації файлів
4. Якщо дата стара, завантажте файли знову

## Швидкий чеклист

- [ ] Очистив кеш браузера (`Ctrl + Shift + R`)
- [ ] Очистив кеш WordPress
- [ ] Перевірив, що тема активна
- [ ] Перевірив версію теми (1.0.3)
- [ ] Спробував режим інкогніто
- [ ] Перевірив консоль на помилки
- [ ] Перевірив Network на завантаження файлів

## Контакти

Якщо проблема залишається, надішліть скріншот консолі браузера (F12 > Console) розробнику.
