# Мінімалістичний дизайн сторінки кошика WooCommerce

## Що було зроблено

### 1. Створено кастомні шаблони

- `woocommerce/cart/cart.php` - головний шаблон сторінки кошика
- `woocommerce/cart/cart-totals.php` - шаблон підсумків замовлення

### 2. Додано CSS стилі

Мінімалістичний дизайн включає:

- Чистий білий фон для елементів кошика
- Сірий фон (#f8f8f8) для загальної сторінки
- Простий і зрозумілий layout
- Адаптивний дизайн для мобільних пристроїв
- Плавні анімації та переходи
- Кнопки з hover ефектами

### 3. Додано JavaScript функціональність

- Автоматичне оновлення кошика при зміні кількості
- Кнопки + та - для зміни кількості товарів
- Плавна анімація видалення товарів
- Валідація введення кількості
- Стан завантаження для кнопок

### 4. Основні особливості дизайну

#### Порожній кошик

- Велика іконка кошика
- Чіткий заголовок "Your cart is empty"
- Кнопка "Continue Shopping"

#### Список товарів

- Зображення товару (120x120px на десктопі)
- Назва товару з посиланням
- Ціна за одиницю
- Поле кількості з кнопками +/-
- Загальна вартість
- Кнопка видалення

#### Підсумки замовлення

- Чистий білий блок справа
- Проміжний підсумок
- Доставка (якщо потрібно)
- Податки (якщо потрібно)
- Загальна сума
- Велика кнопка "Proceed to Checkout"

### 5. Адаптивність

#### Десктоп (>1024px)

- Товари в таблиці з усіма колонками
- Підсумки справа (max-width: 450px)

#### Планшет (768px - 1024px)

- Ціна за одиницю прихована
- Показується тільки загальна вартість

#### Мобільний (<768px)

- Вертикальний layout
- Зменшені зображення (80x60px)
- Всі елементи один під одним
- Підсумки на всю ширину

## Як використовувати

1. Файли вже створені в темі `kacosmetics`
2. Перейдіть на сторінку кошика вашого сайту
3. Дизайн автоматично застосується

## Налаштування

### Зміна кольорів

Відредагуйте файл `style.css` і знайдіть секцію:

```css
/*--------------------------------------------------------------
# WooCommerce Cart Page - Minimalist Design
--------------------------------------------------------------*/
```

Основні кольори:

- Чорний: `#000` - кнопки, текст
- Рожевий: `#FF4586` - hover стан кнопок
- Сірий фон: `#f8f8f8`
- Білий: `#fff` - картки товарів

### Зміна розмірів

- Максимальна ширина контейнера: `.cart-container { max-width: 1200px; }`
- Розмір зображень: `.cart-item-image { width: 120px; height: 120px; }`

## Додаткові можливості

### Автоматичне оновлення

Кошик автоматично оновлюється через 800мс після зміни кількості товару.

### Анімації

- Плавне видалення товарів
- Hover ефекти на кнопках
- Smooth scroll до повідомлень

### Доступність

- ARIA labels для кнопок
- Підтримка клавіатури
- Focus states для всіх інтерактивних елементів

## Troubleshooting

### Стилі не застосовуються

1. Очистіть кеш браузера
2. Очистіть кеш WordPress (якщо використовується плагін кешування)
3. Перевірте, чи активна тема `kacosmetics`

### JavaScript не працює

1. Перевірте консоль браузера на помилки
2. Переконайтеся, що jQuery підключений
3. Очистіть кеш

### Проблеми з адаптивністю

1. Перевірте viewport meta tag в header.php
2. Протестуйте в режимі інкогніто
3. Очистіть кеш браузера

## Контакти

Якщо виникли питання або потрібна допомога, зверніться до розробника теми.
